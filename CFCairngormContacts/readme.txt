APPLICATION: CFCairngormContacts
--------------------------------
A simple Flex-based contact manager built using the Cairngorm Framework.


INSTALLATION
-------------------------
Instructions for installing and setting up the application are available at:
http://code.google.com/p/cfcairngormcontacts/wiki/Installation


CHANGELOG
---------

23 Febuary 2007 (1.0): First official release
